# BEBE HEIST

A peekaboo stealth game. Rob the bank. Don't let the babies see your face.

## Setup

```
npm install
npm run dev
```

## Controls

- **WASD** -- Move
- **SHIFT** -- Sprint (noisy!)
- **SPACE** (hold) -- Peekaboo (hide your face)
- **Click** -- Throw cheese (stuns babies)
- **E** -- Interact / Search / Doors
- **Q / HOLD Q** -- Use tool / Tool wheel
- **ESC** -- Pause
- **R** -- Restart
