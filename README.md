# BEBE HEIST

A peekaboo stealth game. Rob the bank. Don't let the babies see your face.

## Setup

```
npm install
```

## Development

```
npm run dev
```

Opens a local dev server with hot reload at `http://localhost:5173`.

## Production Build

```
npm run build
npm run preview
```

## Controls

- **WASD** -- Move
- **SPACE** (hold) -- Peekaboo (hide your face, limited breath)
- **Click** -- Throw cheese (stuns babies)
- **E** -- Grab loot
- **Q** -- Use tool (iPad, TV Remote, Pacifier)
